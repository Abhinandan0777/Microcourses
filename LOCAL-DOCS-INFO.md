## 🔒 Why Keep Docs Local?

1. **Security**: Some docs contain deployment details and architecture info
2. **Cleaner Repo**: GitHub repo stays focused on code
3. **Privacy**: Your notes and summaries stay private
4. **Flexibility**: Update docs freely without cluttering Git history
